### fix702 ‚Äî Phase 1: top tab bar (first VISIBLE change)
`tv_top_tab_bar.dart` `_TabButton`: bespoke StatefulWidget (custom FocusNode +
600ms held-OK timer + flat yellow `Border.all`) ‚Üí **`TvFocusable`**. Gains the
accent focus ring (instant-in/120ms-out ‚Äî no focus flash) + 1.05√ó lift. The
selected pill keeps its **section-identity color** (our win). Held-OK is wired
only for long-press tabs (`onHeldOk: onLongPress`); for the rest `onHeldOk` is
null so a held OK still fires `onTap` = switch (preserves fix607). 600ms hold
kept (not the 500ms default). The tab bar is a `!hasTouchScreen` surface, so
`TvFocusable`'s touch `onLongPress‚ÜíonHeldOk` is unreachable on touch devices ‚Äî
the old fix607 touch-footgun guard is moot.
