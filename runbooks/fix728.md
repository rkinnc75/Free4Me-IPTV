# fix728 — Inter type face on TV (mock §2, 10-foot type) — v4.1.32

TV only; phone/touch UI keeps the platform default (byte-identical). Task #47.
The last of the owner-approved "top visual payoff" items.

## What
Bundles Inter as the app-wide TV font (mock §2 "10-foot type"). The rest of the
redesign (micro-ramp sizes, accent rings, glass) was built on the default face;
Inter is the final layer that makes the whole TV UI read as designed.

## How
- `assets/fonts/Inter.ttf` — the OFL 1.1 variable font from github.com/google/fonts
  (`Inter[opsz,wght].ttf`, ~856 KB). Its `wght` axis maps to `TextStyle.fontWeight`
  automatically, so every weight comes from the one file. License shipped at
  `assets/fonts/Inter-OFL.txt`.
- `pubspec.yaml`: `fonts: - family: Inter` declaration (the `assets/` glob already
  packages the file; the family block is what makes `fontFamily: 'Inter'` resolve).
- `lib/main.dart`: `ThemeData(fontFamily: hasTouchScreen ? null : 'Inter', …)` —
  TV only. Same `hasTouchScreen` gate the fix707 accent chrome uses.
