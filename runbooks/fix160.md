## fix160.1 — `persistDebugLogging` → `updateSettings()` (ERROR, line 1074)

**File:** `lib/settings_view.dart`

The analyzer's "enable logging" branch calls a method that doesn't exist. The
surrounding lines already set the logger + state correctly; only the persist
call is wrong.

**Current (lines 1072–1076):**
```dart
      if (enable == true) {
        await AppLog.setEnabled(true);
        await SettingsService.persistDebugLogging(true);
        if (mounted) setState(() => settings.debugLogging = true);
      }
```
**Replace with:**
```dart
      if (enable == true) {
        await AppLog.setEnabled(true);
        if (mounted) setState(() => settings.debugLogging = true);
        updateSettings();   // fix160: persist via the standard helper (no persistDebugLogging method exists)
      }
```

> `updateSettings()` (settings_view.dart:913 → `SettingsService.updateSettings(
> settings)`) persists the `debugLogging` flag with all other settings — the
> same path the working "Enable debug logging" switch uses. There is no
> dedicated `persistDebugLogging`.

## fix160.2 — Duplicate `trailing` in `_bufferSlider` (ERROR, line 1354)

**File:** `lib/settings_view.dart`

fix156.2c added `trailing: _helpIcon(...)` (line ~1344) but the slider row
already had `trailing: SizedBox(... value readout ...)` (line ~1354) → two
`trailing:` on one `ListTile`. Merge both into a single `trailing: Row`.

**Current (`_bufferSlider`, lines ~1341–1362):**
```dart
    return ListTile(
      // fix156: plain text title so the row body is the D-pad target;
      // help icon moves to trailing as a separate focus stop.
      title: Text(label),
      trailing: _helpIcon(title: help.title, body: help.body),
      subtitle: _DpadFriendlySlider(
        value: value.clamp(min, max),
        min: min,
        max: max,
        divisions: divisions,
        label: value.round().toString(),
        onChanged: onChanged,
      ),
      trailing: SizedBox(
        width: 56,
        child: Text(
          value.round().toString(),
          textAlign: TextAlign.right,
          style: const TextStyle(fontWeight: FontWeight.w600),
        ),
      ),
    );
```
**Replace with:**
```dart
    return ListTile(
      // fix156/160: plain text title so the row body is the D-pad target.
      title: Text(label),
      subtitle: _DpadFriendlySlider(
        value: value.clamp(min, max),
        min: min,
        max: max,
        divisions: divisions,
        label: value.round().toString(),
        onChanged: onChanged,
      ),
      // fix160: ONE trailing — value readout + help icon together (both
      // focus-reachable on D-pad; no duplicate `trailing`).
      trailing: Row(
        mainAxisSize: MainAxisSize.min,
        children: [
          SizedBox(
            width: 56,
            child: Text(
              value.round().toString(),
              textAlign: TextAlign.right,
              style: const TextStyle(fontWeight: FontWeight.w600),
            ),
          ),
          _helpIcon(title: help.title, body: help.body),
        ],
      ),
    );
```

> Remove BOTH old `trailing:` lines; replace with the single `trailing: Row`.
> Title stays plain text (the 156.2c focus-steal fix). The `_DpadFriendlySlider`
> in `subtitle` remains the adjustable control.
