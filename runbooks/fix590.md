## Change

- `android/app/build.gradle` release block: `minifyEnabled true`,
  `shrinkResources true`, `proguardFiles getDefaultProguardFile(
  'proguard-android-optimize.txt'), 'proguard-rules.pro'`.
- New `android/app/proguard-rules.pro`. FIRST PASS uses `-dontobfuscate`
  (shrink-only, no renaming) so any crash cannot be a renamed-symbol mismatch.
  `-keep` rules cover everything R8 can't see a static reference to:
  - manifest/reflection entry points: `Free4MeApplication`, `MainActivity`,
    `CastPlugin`, `CastOptionsProvider`;
  - JNI: `-keepclasseswithmembernames class * { native <methods>; }`;
  - Flutter embedding + `GeneratedPluginRegistrant`;
  - reflection/manifest plugins: workmanager, flutter_foreground_task,
    package_info_plus, device_info_plus, connectivity_plus;
  - Cast SDK (`OptionsProvider` implementors);
  - media_kit / libmpv bridge (`com.alexmercerind.**`), sqlite3;
  - enum `values()/valueOf()`, Parcelable `CREATOR`.

## Pre-flight

- Local `flutter build apk --release` succeeds (R8 runs, no shrink/config
  error). Local build is debug-signed (no keystore) so it can't `install -r`
  over the CI app ‚Äî runtime is verified on the CI signed APK below.
