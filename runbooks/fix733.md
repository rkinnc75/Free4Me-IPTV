  on the SAME poster spec as Movies/Series (maxExtent 130, AR 0.838, spacing 6),
  reusing ChannelTile verbatim (source-tint edge bar + play/series drill-in +
  isHistory "Remove from history"). No category rail (history is a flat list).
  Empty / error+Retry / loading states. `onRemoveHistory` reloads.
- `tv_shell.dart`: `ViewType.history` now builds `TvHistoryView` (keyed on
  `_historyGen`) in both the initial build and the Clear-history rebuild
  (was phone `Home`).
