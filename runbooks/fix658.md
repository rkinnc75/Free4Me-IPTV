## Resolved in this release

**Dead-code deletions:**
- `lib/menu_tile.dart` (171), `lib/validators.dart` (176),
  `lib/models/snapshot.dart` + `lib/models/stack.dart` (172) — whole files, zero
  importers.

  classes; `XtreamCategory` kept.
- `lib/backend/xtream_epg.dart` (169) — dead `fetchShortEpg` + its private-only
  helpers `_parseEpochSecs` / `_decodeBase64`, plus the now-orphaned
  `dart:convert` / `http_client` / `models/program` imports. `XtreamEpg.xmltvUrl`
  (used at epg_service.dart) kept.

**Dependencies:** dropped unused `logging` (145) and `flutter_svg` + orphan asset
`assets/smpte_color_bars.svg` (146).

**One-liners / small hardening:**
- `setup.dart`: run FormBuilder `validate()` before the submit proceeds so an
  invalid URL is rejected on press (163); `if (!mounted)` guard in `selectFile`
  after the picker await (164).
- `AndroidManifest.xml`: remove the duplicate `WAKE_LOCK` permission (165).
- `conn_timing.dart`: native `Socket.connect(timeout:)` so a timed-out connect
  is cancelled, not leaked; TLS note (166).
- `epg_discovery/epg_validator.dart`: gzip bodies are now INFLATED and run
  through the same `<?xml`/`<tv`/`<programme>` checks instead of being trusted
  on magic bytes alone (167). Old "trusts the loader" test replaced with three
  cases (rejects gzipped non-XMLTV, accepts gzipped real XMLTV, rejects a fake
  gzip stream).
- `search_perf_test.dart`: documents that the cold-cache PRAGMA runs on the
  write connection while probes use the read pool — accepted/caveated per the
  spec decision, no rearchitecture (168).
- `edit_dialog.dart`: `if (!mounted)` guard before `setState` after the
  color-picker await (170).
- `player/player_key_action.dart`: alias discrete remote transport keys
  (mediaTrackPrevious/Next, mediaRewind/FastForward) to the existing
  channelUp/Down + seek actions — no enum or caller change (174).
- `tv/tv_shell.dart`: move `autofocus` from the destructive Clear button to
  Cancel in the clear-history dialog (175).
- `whats_new_modal.dart`: wrap the dialog in `PopScope` so `lastSeenVersion` is
  recorded on ANY dismissal (Back/barrier/button), guarded to write once (177).
- `widgets/dpad_text_field.dart`: remove the dead `initialValue` field + ctor
  param, zero callers (178).
- `test/export_zip_test.dart`: rename the existing test to an archive-mechanism
  smoke test + add a credential-exclusion contract test (179).
